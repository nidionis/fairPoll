{% trans "How it works:" %}

  • {% trans "1. Gather your voters: All voters must belong to the House." %}
  • {% trans "2. Create a house: Set it up with all its voting members." %}
    {% trans "Note: To prevent ballot stuffing, you cannot add new voters after the House is created." %}
  • {% trans "3. Create the Poll: Initiate a new poll from within the House." %}
  • {% trans "4. Distribute Tickets: The poll creator must print, cut out, and randomly distribute the voting tickets (for example, by having members draw them from a box)." %}
  • {% trans "5. Cast Votes: Each voter joins the poll using the poll ID, and uses their unique ticket to cast their vote anonymously (no sign-in required when voting a ticket poll)." %}
  • {% trans "6. Review Results: Once voting ends, the results are stored in the House archives. The complete ballot can be downloaded for independent verification." %}
